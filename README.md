# 🎮 Commands

```text
!sr music          → Add a song
!song              → Current song
!next              → Next YTM song
!nextsr            → Next local SR
!playlist          → Show SR queue
!skip              → Skip current song
!prio music        → Skip Prio
!clear             → Clear all queue
```
