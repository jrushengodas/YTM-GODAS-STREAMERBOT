# 🎧 CONFIG YOUTUBE MUSIC DESKTOP

## 1. Lancer l’application

Ouvrir :

```text
YouTube Music Desktop.exe
```

---

## 2. Activer le serveur API

Dans YouTube Music Desktop :

```text
Paramètres → Extensions → Serveur API
```

Configurer :

```text
Activé : Oui
Hôte : 127.0.0.1
Port : 26538
Autorisation : Pas d’autorisation
```

---

## 3. Redémarrer l’application

Après modification, redémarrer YouTube Music Desktop.

---

## 4. Tester l’API

Ouvrir dans le navigateur :

```text
http://127.0.0.1:26538/swagger
```

Si la page s’ouvre, l’API fonctionne.

---

## 5. Valeurs recommandées

```text
ytmHost = 127.0.0.1
ytmPort = 26538
```
